# CHARISMA
A Character-Based, Behavior-Informed Framework for Simulating Personality-Driven Social Interaction with Multi-LLM Agents
## prerequirements
We use `uv` as a package manager in the project. Click [here](https://docs.astral.sh/uv/getting-started/installation/) to install it 
## app
To run the app, please use the following command: `uv run -m streamlit run charisma`.
Mind the model/provider defined, as the free deepseek version is sometimes not available.
